description: Git commit the files changed during this development work in multiple batches
---

Git commit the files changed during this development work in multiple batches, follow these rules:

1. **Language**: Use English only
2. **Format**: Keep it brief, one line only
3. **Example**: `feat: add user authentication` or `fix: resolve payment timeout issue`
4. **scope**: Only commit files in current task changed
5. **Granularity**: Commit small, focused changes separately; one logical change per commit
